
# 📊 Telco Customer Churn Analysis

## 🎯 Problem Statement
Identify which customers are likely to churn and why, 
so the business can take action to retain them.

## 🛠️ Tools Used
- Excel — Data Cleaning
- SQL (MySQL) — Data Analysis
- Power BI — Dashboard & Visualization

## 📌 Key Insights
1. Overall churn rate is 26.54%
2. Month-to-month customers churn 42% vs 3% for 2-year contracts
3. New customers (0-1 year) have highest churn rate
4. Electronic check users churn more than other payment methods
5. Fiber optic users churn more despite faster internet
6. Revenue lost due to churn: $139K/month

## 💡 Business Recommendations
1. Offer discounts to convert month-to-month to annual contracts
2. Focus retention efforts on customers in first 12 months
3. Investigate why fiber optic customers are dissatisfied
4. Promote auto-payment methods over electronic checks
